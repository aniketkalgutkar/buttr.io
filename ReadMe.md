This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Prereq
- Node Version: 20+

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

# Blog content
 - The content for the blog goes in the `content` folder
 - The assets for the blog like images goes in the public/content_assets folder



# Build for GitHub

```
npm run gh-build
```
Copy all content from the /out folder and copy it to buttr.io's gh-pages branch. The index.html is on the gh-pages branch will be picked as the root page


# Keywords optimization


Run a Google Trends search for "GEO SEO" and "Generative Engine Optimization" to see which countries/regions are leading the interest.

Use AlsoAsked.com to find the specific questions people ask about brand visibility on LLMs.

Create a "Seed List" of these questions and put them into Google Keyword Planner to get the hard numbers.
